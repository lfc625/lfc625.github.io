lodash常见方法

Array

_.chunk(array, [size=1])

将array拆分多个 size 长度的块，把这些块组成一个新数组

_.compact(array)

创建一个新数组并包含原数组中所有的==非假值==元素。例如 false、null、 0、””、undefined 和 NaN 都是“假值”。

_.difference(array, [values])

过滤数组

_.drop(array, [n=1])

将 array 中的前 n 个元素去掉，然后返回剩余的部分。

_.dropRight(array, [n=1])

将 array 中的后 n 个元素去掉，然后返回剩余的部分。
