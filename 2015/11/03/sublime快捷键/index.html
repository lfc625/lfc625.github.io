sublime常用快捷键总结

By liaofeic

2015-11-03 更新日期:2016-01-11

文章目录

  2. 2. 像极客一样使用Sublime Text

Sublime Text快捷键：

Ctrl+Shift+P：打开命令面板
Ctrl+P：搜索项目中的文件
Ctrl+G：跳转到第几行
Ctrl+W：关闭当前打开文件
Ctrl+Shift+W：关闭所有打开文件
Ctrl+Shift+V：粘贴并格式化
Ctrl+D：选择单词，重复可增加选择下一个相同的单词
Ctrl+L：选择行，重复可依次增加选择下一行
Ctrl+Shift+L：选择多行
Ctrl+Shift+Enter：在当前行前插入新行
Ctrl+X：删除当前行
Ctrl+M：跳转到对应括号
Ctrl+U：软撤销，撤销光标位置
Ctrl+J：选择标签内容
Ctrl+F：查找内容
Ctrl+Shift+F：查找并替换
Ctrl+H：替换
Ctrl+R：前往 method
Ctrl+N：新建窗口
Ctrl+K+B：开关侧栏
Ctrl+Shift+M：选中当前括号内容，重复可选着括号本身
Ctrl+F2：设置/删除标记
Ctrl+/：注释当前行
Ctrl+Shift+/：当前位置插入注释
Ctrl+Alt+/：块注释，并Focus到首行，写注释说明用的
Ctrl+Shift+A：选择当前标签前后，修改标签用的
F11：全屏
Shift+F11：全屏免打扰模式，只编辑当前文件
Alt+F3：选择所有相同的词
Alt+.：闭合标签
Alt+Shift+数字：分屏显示
Alt+数字：切换打开第N个文件
Shift+右键拖动：光标多不，用来更改或插入列内容
鼠标的前进后退键可切换Tab文件
按Ctrl，依次点击或选取，可需要编辑的多个位置
按Ctrl+Shift+上下键，可替换行

像极客一样使用Sublime Text

选中光标所占的文本

Ctrl + D 
再次Ctrl + D则会选中下一个相同的文本。
一次性选择全部的相同文本进行同时编辑

Alt+F3 
选中文本并按下快捷键，即可一次性选择全部的相同文本进行同时编辑。简单来说，是Ctrl + D的升级版

Ctrl+L
选中整行，继续操作则继续选择下一行，效果和Shift + ↓ 效果基本一样。但shift + ↓ 是从当前光标位置开始选择至行末！
在每行行尾插入光标

Ctrl+Shift+L
先选中多行，(也就是用鼠标稍微在文本拉一下)再按下快捷键，此时，会在每行行尾插入光标，即可同时编辑这些行。
选择括号内的内容

Ctrl+Shift+M
function test(src) {
    eval(src); 
    return y;
}
被认为是括号的结构，() , { } 等
选择括号内的内容（继续选择父括号）。
比如我将光标停留在eval函数里面，
按下快捷键，会先选中代码eval(src)中的src,
然后在按一下，会从内到外选择{ }里面的内容！
应用：

快速选中删除函数中的代码
重写函数体代码
重写括号内里的内容。
在括号内结束或开始位置来回移动

Ctrl+M
在这两个位置来回切换！
在上、下一行插入新行

Ctrl+Enter # **即使光标不在行尾**，也能快速向下插入一行
Ctrl+Shift+Enter # 同上，只不过向相反方向操作
折叠与展开代码

Ctrl+Shift+[ 
操作：选中代码，按下快捷键，
Ctrl+Shift+]
操作： 选中代码，按下快捷键，展开代码。
Ctrl+K+0 
操作：一次性展开所有折叠代码。
快速移动光标

单位性： 我感觉像是一个单词一个单词这样跳过
Ctrl+← #　向左单位性地移动光标，快速移动光标。
Ctrl+→ #　向右单位性地移动光标。
选择文本

shift+↑ # 向上选中多行。
shift+↓ # 向下选中多行。
Shift+← # 向左选中文本。
Shift+→ # 向右选中文本。
移动光标并选择文本

Ctrl+Shift+← # 向左单位性地选中文本。
Ctrl+Shift+→ # 向右单位性地选中文本。
代码位置切换

Ctrl+Shift+↑ # 将光标所在行和上一行代码互换（将光标所在行插入到上一行之前）。
Ctrl+Shift+↓ # 将光标所在行和下一行代码互换（将光标所在行插入到下一行之后）。
Ctrl+Alt+↑ # 向上添加多行光标，可同时编辑多行。
Ctrl+Alt+↓ # 向下添加多行光标，可同时编辑多行。
编辑类

合并选中的多行代码为一行

Ctrl+J
合并选中的多行代码为一行
应用：

将多行格式的CSS属性合并为一行。
复制光标所在整行,并插入到下一行

Ctrl+Shift+D
缩进

Tab # 向右缩进。
Shift+Tab # 向左缩进。
删除代码

Ctrl+K+K # 从光标处开始删除代码至行尾
Ctrl+Shift+K # 删除整行
注释

Ctrl+/  # 注释单行
Ctrl+Shift+/  # 注释多行
转换大小写

Ctrl+K+U  # 转换大写 (upper)
Ctrl+K+L  # 转换小写 (lower)
其他命令...

Ctrl+Z  # 撤销。
Ctrl+Y  # 恢复撤销。
Ctrl+U # 软撤销，感觉和 Gtrl+Z 一样
Ctrl+F2 # 设置书签
Ctrl+T  # 左右字母互换。
F6 单词检测拼写
搜索类

查找

Ctrl+F # 打开底部搜索框。
Ctrl+shift+F # 在文件夹内查找，比如可以在整个项目中查找。
Ctrl+P #打开搜索框。
应用：

输入当前项目中的文件名，快速搜索文件，比如我想跳转到index.html,我稍微输入index也就可能找到相应的文件
输入@和关键字，查找文件中函数名，与按下快捷键Ctrl + R效果一样
输入：和数字，跳转到文件中该行代码，与按下快捷键Ctrl + G效果一样
输入#和关键字，查找变量名,与按下快捷键Ctrl + :效果一样
打开命令框

Ctrl+Shift+P
应用：

打开命名框，输入关键字，调用sublime text插件的功能
安装插件
ESC

Esc 
应用：

退出光标多行选择
退出搜索框
命令框
...
显示类

切换窗口

Ctrl+Tab # 切换当前窗口的标签页。
Ctrl+PageDown # 向左切换当前窗口的标签页。
Ctrl+PageUp # 向右切换当前窗口的标签页。
窗口分屏

Alt+Shift+1 # 窗口分屏，恢复默认1屏（**非小键盘的数字**）
Alt+Shift+2 # 左右分屏-2列
Alt+Shift+3 # 左右分屏-3列
Alt+Shift+4 # 左右分屏-4列
Alt+Shift+5 # 等分4屏
Alt+Shift+8 # 垂直分屏-2屏
Alt+Shift+9 # 垂直分屏-3屏
其他命令...

Ctrl+K+B 开启/关闭侧边栏。
F11 全屏模式
Shift+F11 免打扰模式
